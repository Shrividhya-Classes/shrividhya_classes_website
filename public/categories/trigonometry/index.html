Trigonometric Functions with Negative Angles

Find the value of the following trigonometric expressions:

  • cos(-180°)
  • sin(-90°)
  • tan(-270°)

This is my content.

Solution:

  1. The value of $\cos(-180^{\circ})$ is: $$ \cos(-180^{\circ}) = \cos(180^{\circ}) = -1 $$

  2. The value of $\sin(-90^{\circ})$ is: $$ \sin(-90^{\circ}) = -\sin(90^{\circ}) = -1 $$

  3. The value of $\tan(-270^{\circ})$ is: $$ \tan(-270^{\circ}) = \text{undefined} $$
